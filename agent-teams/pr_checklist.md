# PR Checklist (Agent Team)

## Mandatory
- [ ] Task has an owner lock in `task_queue.yaml`
- [ ] Handoff contract file exists and validator passes
- [ ] QA gate status is `pass` (or coordinator waiver recorded)
- [ ] Verification commands executed and evidence attached
- [ ] No critical defects open

## Quality
- [ ] Scope matches task objective
- [ ] Regression risk assessed
- [ ] Rollback plan noted

## Merge Decision
- [ ] **READY TO MERGE**
- [ ] **HOLD** (include reason)
